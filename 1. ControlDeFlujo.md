# Control de Flujo

> [!NOTE]
> El control de flujo son las estructuras que definen el orden en que se ejecutan las instrucciones de un programa, permitiendo tomar decisiones y repetir bloques de c√≥digo.


# Valores booleanos

| Valor | Descripci√≥n |
| --- | --- |
| `True` | representa el valor verdadero de una expresi√≥n |
| `False` | representa el valor falso de una expresi√≥n |

## if

<img src="https://www.notion.so/icons/light-bulb_yellow.svg" alt="https://www.notion.so/icons/light-bulb_yellow.svg" width="40px" />

Una sentencia if puede definirse como: si X condici√≥n es verdadera, se ejecuta el c√≥digo de la clausula.

</aside>

Sintaxis

```python

    # bloque a ejecutar
```

## else

<aside>
<img src="https://www.notion.so/icons/light-bulb_yellow.svg" alt="https://www.notion.so/icons/light-bulb_yellow.svg" width="40px" />

La clausula else solo se ejecuta cuando la condici√≥n if es `false`

</aside>

```python

    # se ejecuta si es True

    # se ejecuta si la condici√≥n de arriba es False
```

## while

<aside>
<img src="https://www.notion.so/icons/light-bulb_yellow.svg" alt="https://www.notion.so/icons/light-bulb_yellow.svg" width="40px" />

Se utiliza para repetir un bloque de c√≥digo mientras una condici√≥n sea verdadera (True)

</aside>

Sintaxis

```python
while condicion:
    # bloque de c√≥digo que se ejecutar√° mientras la condici√≥n se True
    
    
# Si la condici√≥n es True, el bloque se ejecuta
# Si la condici√≥n es False, el bucle termina y el flujo contin√∫a con la sig instrucci√≥n
```

### break & continue

<aside>
<img src="https://www.notion.so/icons/light-bulb_yellow.svg" alt="https://www.notion.so/icons/light-bulb_yellow.svg" width="40px" />

**break ‚Üí** rompe el bucle inmediatamente

**continue** ‚Üí salta el resto del bloque y pasa a la siguiente iteraci√≥n

</aside>

```python
x = 0
while x < 10:
    x += 1
    if x == 5:
        continue  # salta el n√∫mero 5
    if x == 8:
        break     # detiene el bucle completamente
    print(x)
```

### argumentos range()

```python
'''
**rango de 2 argumentos** 

iniciar√° iterando el primer argumento y el segundo es el final, 
vamos a iterar de 10 al 20
'''
for n in range(10,20)
    print(n)
  
'''
**rango con 3 argumentos** 
El 1er y 2do argumento son los valores inicial y final, el 3er argumento "step" 
"step" es la cantidad en que incrementa la variable despues de cada iteraci√≥n
'''

for n in range(10,20,2) # itera del 10 al 20 de 2 en 2 12,14,16,18
    print(n)
    

# argumentos negativos

for n in range(10,-1,-1)
    # bloque a ejecutar
    
```
